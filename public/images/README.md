# Haveli Main Page Slider Images

To display your custom images on the homepage slider of the **Haveli Banquet Hall And Restaurant** website, follow these simple guidelines:

1. Place your **5 desired images** inside this exact folder (`public/images/`) of your repository.
2. Name them exactly as follows:
   - `image1.jpg`
   - `image2.jpg`
   - `image3.jpg`
   - `image4.jpg`
   - `image5.jpg`

3. Push your changes to GitHub or deploy. The website is configured to automatically load these local images!
4. If any image is missing, the system will gracefully fall back to a high-quality royalty-free restaurant/banquet design placeholder so the site remains stunning.
